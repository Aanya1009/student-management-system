# Student Management System (Java - Console Based)

# Project Description
This project is a simple Student Management System developed using Java.  
It is a console-based application that allows users to manage student records such as adding, viewing, searching, and deleting student details.

This project was developed as part of Java coursework to demonstrate OOP concepts, exception handling.

# Features
- Add new student records  
- Display all students  
- Search student by ID  
- Delete student record  
- Automatic grade calculation  
- Save data to file (`students.txt`)  
- Load data from file on startup  

# Technologies Used
- Java (JDK 8 or above)  
- OOP concepts  
- File Handling  
- Collections  

# System Requirements
- Java JDK (version 8 or above)
- Terminal / Command Prompt

Check installation:
java -version  
javac -version

# How the Program Works
The program uses a menu-driven approach. The user selects options to perform operations like adding, viewing, searching, and deleting student records. 
Student data is stored using an ArrayList during runtime and can be saved to a file for future use.

# Error Handling
The program handles invalid inputs using exception handling. For example, if the user enters a non-numeric value where a number is expected, the program displays an error message instead of crashing.
